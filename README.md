# 🏏 IPL Exploratory Data Analysis

An end-to-end EDA project on the Indian Premier League (IPL) dataset using Python.

## 📁 Project Structure
```
ipl_eda_project
│
├── data
│   ├── matches.csv
│   └── deliveries.csv
│
├── notebook
│   └── ipl_analysis.ipynb
│
└── README.md
```
## 🛠️ Libraries Used
- pandas
- numpy
- matplotlib
- seaborn

## 🚀 How to Run
1. Clone the repo
2. Install dependencies: `pip install pandas numpy matplotlib seaborn jupyter`
3. Open `notebook/ipl_analysis.ipynb`
